## Instructions
To commit to GitHub, use the following commands:

### Check GitHub access:
``` bash
ssh -T github-jono
```
should return
``` bash
Hi jonathan-huang! You've successfully authenticated, but GitHub does not provide shell access.
```
### Git commands
``` bash
git remote -v   # checks remote ssh status
git status   # shows current Git status and changes to be committed
git log   # shows commit history
git add   # add changes
git commit -m "commit message"   # commits to GitHub, should see changes on GitHub webpage
```
